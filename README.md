## 📌 作品簡介

本專題為網頁設計課程之期末實作成果，從基礎前端網頁架構出發，延伸實作出一套具備前後台管理功能的簡易賣場網站。以「電競次元商城」為主題，完整模擬電商平台的購物流程，並加入後台管理系統，讓管理員能即時維護賣場內容。

## 🖥️ 功能說明

### 🏠 首頁（index.html）
- 展示最多 5 件「熱門推薦商品」卡片，由後台管理員自行標記推薦
- 提供搜尋欄，輸入關鍵字後跳轉至商品頁並自動過濾結果
- 支援自訂 Logo 顯示（從 `localStorage` 讀取）

---

### 🛍️ 商品頁（page1.html）
- 顯示所有上架商品，包含圖片、名稱、價格與庫存數量
- 支援關鍵字搜尋，即時過濾商品列表
- 點擊「加入購物車」將商品加入 `localStorage` 購物車
- 商品庫存為 0 時，加入按鈕自動停用

---

### 🛒 購物車（page3.html）
- 列出購物車內所有商品，支援勾選要結帳的項目
- 可調整各商品數量（不得超過庫存上限）
- 顯示勾選商品的即時總金額
- 提供「下單」功能，扣除對應庫存並清除已購商品
- 支援「清空購物車」一鍵清除所有項目

---

### 📖 關於我們（page4.html）
- 顯示從後台編輯儲存的「關於我們」文字內容
- 嵌入浮動式「聯繫我們」對話框，使用者可輸入訊息與商家互動

---

### ⚙️ 賣場管理後台（page2.html）
需登入管理員帳號才可使用，預設帳號與密碼如下：

| 帳號 | 密碼 |
|------|------|
| `mystore` | `412410077` |

登入後可進行以下操作：

1. **帳號密碼修改** — 輸入舊密碼後設定新帳號與密碼，並重新登入生效
2. **Logo 更換** — 上傳自訂圖片作為網站 Logo，或還原為預設 Logo
3. **商品管理**
   - 新增商品（名稱、價格、圖片上傳）
   - 調整各商品庫存數量
   - 設定最多 5 件商品為「熱門推薦」，顯示於首頁
   - 刪除商品
   - 清空所有商品 / 還原為預設商品
4. **關於我們編輯** — 即時編輯「關於我們」頁面的文字內容，並可還原為預設文字

## 💾 資料儲存機制

本專案全程使用瀏覽器 `localStorage` 進行前端資料持久化，無需後端伺服器。

| 鍵值 | 說明 |
|------|------|
| `products` | 所有商品資料（名稱、價格、庫存、圖片、推薦狀態） |
| `cart` | 購物車內容 |
| `adminAccount` | 管理員帳號與密碼 |
| `customLogo` | 自訂 Logo 的 Base64 圖片資料 |
| `aboutUs` | 關於我們頁面的文字內容 |
| `recommendedProductIds` | 熱門推薦商品的 ID 清單 |
| `isLoggedIn` | 管理員登入狀態 |
| `cleared` | 標記商品是否已被手動清空 |

> ⚠️ 注意：`localStorage` 資料儲存於使用者的瀏覽器本地端，清除瀏覽器快取將導致資料遺失。
